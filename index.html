Requirements

  • 🪟
    Windows 10 or 11 64-bit recommended
  • 🎵
    Spotify Premium Required The Spotify Web API used by WinIsland does not work on the free plan. Playback control requires Premium.
  • 💡
    Spotify Desktop or Spotify Web Recommended Spotify must be active on at least one device. Fun fact — WinIsland can also control Spotify playing on your phone or any other device on the same account!

Setup Guide

How to get your Spotify Client ID

Create a free Spotify Developer App in about 2 minutes.

1

Go to the Spotify Developer Dashboard

Visit developer.spotify.com/dashboard and log in with your Spotify account.

2

Click "Create App"

You should see a Create App button on the dashboard. If you land on a Documentation or Community page instead, check that the URL ends with /dashboard.

📸 Dashboard showing Create App button
3

Fill in a name and description

Enter any name, description and website (optional) — for example "Dynamic Island". These are just labels for your own reference.

📸 Create App form with name and description
4

Add the Redirect URI

Under Redirect URIs, add the exact URL below. Then click Add.

http://127.0.0.1:8888/callback
5

Select Web API

Under "Which API/SDKs are you planning to use?", check Web API.

📸 API/SDKs selection with Web API checked
6

Save, then copy your Client ID

Click Save. On your new app's page, copy the Client ID — a long string of letters and numbers. Paste it into the WinIsland setup prompt.

Tip: The Client ID is not a secret — it is safe to copy and share.
📸 App page with Client ID
7

Approve the connection

Your browser will open a Spotify authorisation page. Click Agree to allow WinIsland to read your playback and control Spotify on your behalf.

📸 Spotify Agree page
8

You're all set!

The browser tab will close automatically and WinIsland will start showing your Spotify playback. You're good to go!

📸 WinIsland connected and playing
